### 🌿 About me

I like turning messy, manual workflows into something a single click can handle. Most of my projects pair a **Next.js / TypeScript** frontend with a **FastAPI / Python** backend, and run AI locally with Whisper, Ollama, and FFmpeg. No cloud bills required.
